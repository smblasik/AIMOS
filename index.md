# Vault Map

## Structure
```
Root Folder/
├── +/                        # inbox / unsorted; default drop for all new files
├── 1 Projects/               # active project notes; files have project-code: frontmatter
├── 2 Areas/                  # ongoing responsibilities (Grow Self, Others, Firm)
├── 3 Resources/              # reference material and templates
│   └── Templates/            # copy-paste starters
├── 4 Notes/                  # working notes organized by type
│   ├── Meeting Notes/        # dated meeting notes (project-coded)
│   ├── Weekly Updates/       # YYYY-MM-DD Weekly Update.md
│   ├── Topic Notes/          # evergreen / reference notes by subject
│   └── Daily Notes/          # daily log entries (sparse)
├── 5 Archive/                # dead or completed content; ignore by default
└── AIMOS/                     # AI OS system config; do not modify unless asked
```

## Skills

Located in `AIOS/skills/` — each is a subfolder with a `SKILL.md`:

| Skill           | Description                                                                   |
| --------------- | ----------------------------------------------------------------------------- |
| `feedback`      | Draft structured feedback using the SBI (Situation-Behavior-Impact) framework |
| `meeting-notes` | Meeting note structure and formatting                                         |
| `skill-creator` | Create new skills in the correct format                                       |
| `weekly-update` | Synthesize notes, emails, and calendar into a weekly status update            |
| `writing-style` | Tone, structure, and voice guidance                                           |
